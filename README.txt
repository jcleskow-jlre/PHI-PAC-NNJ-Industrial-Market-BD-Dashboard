PA/NJ/DE Industrial Market & BD Dashboard - v13 responsive

GitHub Pages upload instructions:
1. Upload index.html to the root of the GitHub repository.
2. In GitHub, go to Settings > Pages.
3. Set Source to Deploy from a branch.
4. Select main branch and /root folder.
5. Save and wait a few minutes for GitHub Pages to publish.

This package is self-contained. The dashboard data is embedded inside index.html.
The only external dependencies are Leaflet CSS/JS and Esri map tiles loaded from public CDNs.
Internet access is required for the satellite map tiles and Leaflet library to load.
